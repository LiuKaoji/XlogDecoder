### 安装

1.下载源码使用Xcode编译
2. 直接下载release版本程序运行(不推荐 证书失效会闪退)

### 操作说明
```bash
01.单文件操作,可勾选自动打开日志
02.多文件操作,拖拽多个文件同时解压
03.自动检测TRTC调用Enter Room API事件,列表展示
04.基于02和03可以批量导入日志 检测所有进房事件
